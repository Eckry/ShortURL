# Urlsito
## Url shortener built with react
## Technologies
* React
* Turso database

## Installation
Clone the repository and install dependecies: 
```bash
$ git clone https://github.com/Eckry/ShortURL.git
$ cd ShortURL
$ npm install
```

Start development:
```bash
$ npm run dev
```
You can see it here: [https://urlsito.netlify.app/](https://urlsito.netlify.app/)
